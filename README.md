# 🚀 Logic Layer Tasks CLI

A robust, Object-Oriented Command Line Interface (CLI) application built in **Java** to manage and persist daily tasks effectively. This project is built completely from scratch using custom logic and manual loops, without relying on built-in sorting libraries or complex frameworks, showcases core OOP concepts and data structuring.

---

## ✨ Features

- **Full CRUD Operations:** Create, Read, Update, and Delete tasks seamlessly through an interactive terminal menu.
- **Manual Priority Sorting:** Custom implementation of the **Bubble Sort** algorithm to arrange tasks by urgency levels (1 to 5).
- **Chronological Date Sorting:** Custom date sorting using string-based comparison (`.compareTo()`) to sort tasks by their upcoming due dates.
- **Engine Search Engine:** Instant lookup mechanisms to search for specific tasks by their unique ID or by matching title keywords.
- **Data Persistence (Serialization):** Automatically saves your tasks into a binary format file (`tasks_data.ser`) upon exiting and reloads your previous data seamlessly on startup.

---

## 🛠️ Tech Stack & Concepts

- **Language:** Java 8+
- **Paradigm:** Object-Oriented Programming (OOP) (Encapsulation, Blueprint Objects)
- **Data Structures:** Custom dynamic list manipulation and manual loop architectures.
- **I/O Streams:** `ObjectOutputStream` & `ObjectInputStream` for Binary Serialization.
